Go/Gin:
mkdir my-project-name
cd my-project-name
go mod init github.com/yourusername/my-project-name
go get -u github.com/gin-gonic/gin

Then create main.go:
package main

import "github.com/gin-gonic/gin"

func main() {
    r := gin.Default()
    r.GET("/ping", func(c *gin.Context) {
        c.JSON(200, gin.H{"message": "pong"})
    })
    r.Run(":8080")
}

Run with: go run main.go

Go/Gin with GORM & Postgres:
First, install Postgres:
- macOS: brew install postgresql@15 (or latest) then brew services start postgresql@15
- Linux: sudo apt-get install postgresql postgresql-contrib (Ubuntu/Debian) or use your distro's package manager
- Windows: Download from https://www.postgresql.org/download/windows/

Create a database:
createdb mydb
(Or use psql: psql postgres, then CREATE DATABASE mydb;)

Then in your Go project:
go get -u gorm.io/gorm
go get -u gorm.io/driver/postgres

Example connection in main.go:
package main

import (
    "gorm.io/driver/postgres"
    "gorm.io/gorm"
)

func main() {
    // Adjust user/password based on your Postgres setup
    dsn := "host=localhost user=yourusername password=yourpassword dbname=mydb port=5432 sslmode=disable"
    db, err := gorm.Open(postgres.Open(dsn), &gorm.Config{})
    if err != nil {
        panic("failed to connect database")
    }
    // Use db here
}
